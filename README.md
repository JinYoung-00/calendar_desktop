# Yon Calendar & To-Do 🐈‍⬛

귀엽고 심플한 Electron 기반 데스크톱 캘린더 & 투두 리스트 앱입니다.

## ✨ 주요 기능 (Features)

- **📅 캘린더 (Calendar)**: 월별 일정을 한눈에 확인하고 관리할 수 있습니다.
- **📝 투두 리스트 (To-Do List)**: 날짜별 할 일을 기록하고 관리합니다.
- **🎨 다양한 테마 (Themes)**: 
  - 6가지 컬러 테마 지원 (Pink, Yellow, Green, Blue, Purple, Beige)
  - **다크 모드 (Dark Mode)** 지원
- **🖼️ 커스텀 배경**: 투두 헤더에 원하는 이미지를 업로드하여 꾸밀 수 있습니다.
- **📏 반응형 레이아웃**: 윈도우 크기에 따라 투두 섹션의 너비가 유연하게 조절됩니다.
- **🖥️ 커스텀 타이틀바**: 고양이 아이콘이 포함된 깔끔한 커스텀 타이틀바를 제공합니다.
- **💾 자동 저장**: 모든 데이터는 로컬 스토리지에 안전하게 자동 저장됩니다.

## 🚀 시작하기 (Getting Started)

### 사전 요구사항 (Prerequisites)

- Node.js
- npm

### 설치 (Installation)

```bash
git clone https://github.com/your-username/calender.git
cd calender
npm install
```

### 실행 (Run Locally)

개발 모드로 실행하려면 다음 명령어를 입력하세요:

```bash
npm start
```

### 배포 파일 생성 (Build)

설치 파일(Setup.exe)을 생성하려면 다음 명령어를 실행하세요:

```bash
npm run make
```

생성된 파일은 `out/make/squirrel.windows/x64/` 경로에서 확인할 수 있습니다.

## 🛠 사용 기술 (Technologies)

- **Electron**: 데스크톱 앱 프레임워크
- **HTML5 / CSS3 / Vanilla JavaScript**: 프론트엔드 구현
- **Electron Forge**: 패키징 및 배포 도구
